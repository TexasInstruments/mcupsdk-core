# SBL PCIE {#EXAMPLES_DRIVERS_SBL_PCIE_HOME}

SBL PCIe (\ref EXAMPLES_DRIVERS_SBL_PCIE) makes use of a host application running
on another evm (\ref EXAMPLES_DRIVERS_SBL_PCIE_HOST) to send images to the target
EVM for booting.
